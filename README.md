# TaskMind
TaskMind is a modern web application built with .NET 8 and PostgreSQL, following the Onion Architecture principles. 
The platform allows users to manage projects and tasks efficiently, with support for user roles such as Admin, Project Manager, and User.

Key features:

CRUD operations for Projects, Tasks, and Comments

GitHub integration: Import repositories and analyze project insights (active status, complexity, last update)

Role-based access control to restrict actions depending on user roles

Responsive UI built with Bootstrap

API-driven architecture for external data integration

Easily deployable to cloud platforms like Azure or AWS

This project serves as a learning and demonstration platform for modern web development practices, including REST API consumption, identity management, and clean architecture patterns.
